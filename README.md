# 🛠️ homebrew-cli-tools

Welcome to my personal Homebrew tap containing lightweight and useful CLI utilities.

> ⚠️ **Platform Support:** These tools are compiled from source and officially support **macOS** and **Linux** only. Windows is not natively supported.

## 🚀 Getting Started

Follow these simple steps to connect this tap and start using the tools directly in your Terminal. The commands are identical for both macOS and Linux.

### 1. Install Homebrew
If you don't have Homebrew installed yet, run the following command in your Terminal:
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

### 2. Add this Tap
Connect this repository to your local Homebrew system by running:
```bash
brew tap mikhayloboikodev/cli-tools
```

## 📦 Available Tools

Here is the list of available utilities you can install:

### 1. `ftc` (File Text Copy) https://github.com/mikhayloboikodev/CLI-Tool-ftc
This simple CLI tool gives users the opportunity to quickly copy file contents.

* **To install:**
  ```bash
  brew install ftc
  ```
* **Quick Usage:**
  ```bash
  ftc < input.txt
  ```
